# 🤖 Dentcare – AI-Assisted Dental Care Platform

**Dentcare** is a cutting-edge AI-powered platform designed to facilitate dental care consultations and management through a seamless web application. The platform provides real-time consultations, an efficient patient management system, and secure appointment scheduling for both patients and doctors.

- **Live Site**: [Dentcare Live](https://topsoundbd.com/)
- **Server API**: [Dentcare API](https://api.topsoundbdserver.com/)

---

## 🔑 Key Features

### 👤 User Features

- **AI-Powered Voice Assistant**: Engage with an AI-powered voice assistant to discuss dental issues and receive real-time consultations.
- **Appointment Booking**: Schedule appointments with preferred doctors at available times through an easy-to-use system.
- **Secure Transactions**: Integrated Stripe payment gateway to facilitate secure payments for dental services.
- **User Dashboard**:
  - View appointment history and upcoming consultations
  - Manage personal information
  - Track the status of ongoing consultations
- **Consultation History**: Access a record of past consultations, advice, and recommendations from doctors.

---

### 🛠️ Admin and Doctor Features

- **Patient Management**:
  - View and manage patient profiles
  - Track patient consultations and progress
- **Appointment Management**:
  - Approve or reschedule patient appointments
  - View the calendar of upcoming appointments and manage availability
- **Role-Based Access**:
  - Admins can manage doctor and patient roles, assigning permissions and access levels.
- **Admin Dashboard**:
  - Overview of total users, appointments, consultations, and revenue
  - Insights on platform usage and transactions

---

## ⚙️ Technologies Used

### Frontend

- **Next.js** (React-based framework)
- **Tailwind CSS** (Utility-first CSS framework)
- **Radix UI** (UI Component Library)
- **Axios** (For API calls)
- **Date-fns** (Date handling utilities)

### Backend

- **Node.js** (JavaScript runtime)
- **Prisma** (ORM for database management)
- **MongoDB** (NoSQL Database)
- **Cloudinary** (Image storage and management)

### Authentication & Security

- **JWT (JSON Web Tokens)** for secure user authentication
- **Role-Based Access Control** for admin, doctor, and patient roles

### Additional Tools & Libraries

- **React Hook Form** (Form management library)
- **Sonner** (Toast notifications)
- **Recharts** (Data visualization)
- **Zod** (Schema validation)

---

## 📦 API Overview

The backend API handles:

- **Patient and doctor management**
- **Consultation and appointment scheduling**
- **Stripe integration for payments**
- **Role-based access for admins and users**
- **Voice-based consultation powered by AI**

## 📋 Summary

**Dentcare** offers a robust, scalable, and AI-assisted platform for dental care, aiming to improve the consultation experience for patients and operational efficiency for doctors. With features such as real-time voice consultations, secure payments, appointment booking, and comprehensive dashboards for both doctors and patients, it creates a seamless user experience.

The integration of modern technologies like **Next.js**, **Node.js**, **MongoDB**, and **Stripe**, paired with cutting-edge AI capabilities, ensures the platform's reliability, security, and performance, making **Dentcare** a dependable choice for innovative dental care solutions.
